# 川师智发 H5 原型

基于《四川省中小学教师专业发展大模型支持平台》产品文档实现的移动端 H5 单页应用原型。当前版本为纯静态前端，内置 mock 数据，便于后续对接登录、知识库、AI 对话、画像、资源和管理后台 API。

## 功能覆盖

- 登录/注册：账号密码、手机验证码、四类角色切换、首次登录信息完善引导。
- AI 助手：首页欢迎语、快捷问题、知识库/联网/深度思考开关、多轮对话模拟。
- 知识库：容量展示、知识库广场、我的知识库、分类筛选、加入与上传入口。
- AI 工具集市：智能画像、发展路径、资源推送、智能诊断与研修、发展性评价、教研协同、积分成就。
- 管理视图：学校管理员和教育局管理员可看到教师管理、数据报告、资源配置等后台入口。
- 手机端适配：移动优先布局、底部固定导航、安全区适配，桌面端以手机外框展示。

## 本地运行

直接打开 `index.html` 即可预览。也可以在当前目录启动静态服务：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。

测试账号来自产品文档：`abc` / `123456`。
